# Systolic-Array-32x32-Accelerator
A 3-stage pipelined 32x32 Systolic Array Matrix Accelerator implemented via a complete RTL-to-GDSII ASIC design flow on an industrial 32nm cell library. Features 1,024 independent PEs managing an 82k+ register footprint, optimized for AI workloads with zero timing violations under the slow corner.
